## 概要

- Laravel, Inertia.js, Vue.js によるサンプルアプリ
  - 参考：[Laravel Bootcamp - Build Cipher with Inertia](https://bootcamp.laravel.com/inertia/installation)

## 前提条件

- asdf

## 環境

- php 8.4.2
- nodejs 20.10.0

## セットアップ

```
// 依存関係をインストール
asdf install

// Laravel サーバーを起動
php artisan serve

// Vite 開発サーバーを起動
npm run dev
```
